# Mufin
***Bluetooth based IoT connection library***

## Using mufin
### Requirements
- Rust and Cargo
- Nodejs and Npm

### Methods
1. use `npm install @narwhallab/mufin` command to install the project
2. use `@narwhallab/mufin` in nodejs
```js
let addon = require('@narwhallab/mufin')

addon.bluetooth()
```
